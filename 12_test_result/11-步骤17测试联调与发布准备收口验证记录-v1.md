## 6. 发布前仍需目标环境补验

以下项目不是架构口径问题，也不是本机开发阻塞项，但正式发布前必须在目标环境完成：

- Docker 可用环境下补跑 PostgreSQL Testcontainers 集成测试。
- 目标 PostgreSQL 数据库执行 Flyway 迁移并确认版本表状态。
- 配置真实 OpenAI API Key 后验证 LLM、REVIEW、SUMMARY 与 Eval 可选 LLM 评分链路。
- 在同一环境启动前端、后端和数据库，完整走通 `Schema -> Agent -> Workflow 保存/发布 -> Debug -> Runs -> NodeRun 生成 EvalCase -> EvalRun -> 历史对比`。
