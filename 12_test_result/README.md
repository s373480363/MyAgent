## 3. 记录原则

- 测试报告只记录 OpenAI API Key 是否已配置，不记录 Key 明文。
- 测试结论使用 `通过 / 有条件通过 / 不通过`。
- 缺陷按 `P0 / P1 / P2` 分级记录。
- 任何质量闸门不通过，V1 验收结论应为不通过。
