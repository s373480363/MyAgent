## 6. Docker 正式部署复验

```powershell
$env:AGENT_STUDIO_POSTGRES_PASSWORD='agent-studio-dev-pass'
$env:AGENT_STUDIO_OPENAI_API_KEY='local-acceptance-placeholder'
docker compose up -d --build
```

说明：

- 现有 `postgres-data` 卷中保留了旧的数据库口令，导致首次拉起 `api` 容器时数据库鉴权失败

```powershell

docker restart agent-studio-api-1
```
