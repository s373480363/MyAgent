# 8 开发计划总览

## 1. 目录目标

本目录用于沉淀完整 V1 的开发实施方案、里程碑、任务拆解和交付边界。

开发计划必须直接引用 `0_specifications`、`3_product_design`、`4_arch_design`、`6_schema_design`、`7_interface_design` 中已经确认的设计，不允许脱离设计文档另起一套实现口径。
