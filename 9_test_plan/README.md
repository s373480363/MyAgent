# 9 测试计划总览

## 1. 目录目标

本目录用于沉淀 Agent 管理平台 V1 的测试验收计划、验收口径、测试范围、质量闸门和后续测试报告入口。

## 2. 文档清单

| 编号 | 文档 | 说明 |
|------|------|------|
| 01 | `01-Agent管理平台V1测试验收计划-v1.md` | 完整 V1 测试验收计划，覆盖范围、策略、主流程、异常流程、质量闸门和结论口径。 |
| 02 | `02-Agent管理平台V1测试用例清单-v1.md` | 可执行测试用例清单，按环境、Agent、Schema、工作流、接口、运行、节点、Trace、验收、前端和安全边界分组。 |
| 03 | `03-Agent管理平台V1独立测试方案-v1.md` | 独立于既有测试计划的新测试方案，从契约、运行、追溯、UI/API 一致性和性能门槛角度定义验收方法。 |
| 04 | `04-Agent管理平台V1独立测试用例清单-v1.md` | 与独立测试方案配套的可执行用例清单，按合同层、配置层、主数据与版本、运行、追溯、验收和性能分组。 |

## 3. 当前验收口径

- 本次验收以完整 V1 范围全部交付为前提。
- 验收结论使用 `通过 / 有条件通过 / 不通过`。
- 平台本身不包含固定业务逻辑，LLM 相关测试只验证平台流程、结构化输出、错误分类、Trace 和节点验收机制，不预设业务语义效果门槛。
